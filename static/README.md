# Static Files - Logo Images

## Required Images

Please add the following logo images to this folder:

1. **goodwill_gulliver_logo.png** - The combined Goodwill and Gulliver Prep logo (blue background)
2. **onda_verde_logo.png** - The Onda Verde circular logo (green/teal)

## Image Requirements

- Format: PNG (with transparent background preferred)
- Recommended height: 120-150px
- The images will be displayed side-by-side at the top of the form

## How to Add Images

1. Save your logo images with the exact filenames above
2. Place them in this `static` folder
3. The app will automatically display them

## For Azure Deployment

These images will be automatically deployed to Azure when you push to GitHub, as long as they are in the `static` folder and committed to your repository.
